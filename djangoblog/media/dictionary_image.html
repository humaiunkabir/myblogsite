variable name = { Curly braces }

How To Create python Dictionary

Example 1 :-

# dictionary create 
# 2 tarike ke se ham dictionary create kar sakte hai

# 1 method

data  = {'Name' : 'Danish Ali' , 'Age' : 17 }
print(data)

# 2 method

data1 = dict(Name = 'Danish Ali' , age = 17) 
print(data1)

Aap Upper Diye Gye Program ko samanjh sakte hai.

Output :-

{'Name': 'Danish Ali', 'Age': 17}                                                                                             
{'Name': 'Danish Ali', 'age': 17}

Aap upper Diye gye output ko dekh kar program ko samanjh sakte hai.

Dictionary In Python in Hindi

Example 2 :-

data  = {'Name' : 'Danish Ali' , 'Age' : 17 }

for i in data:      # print key
    print(i)

for i in data.values():       # print value
    print(i)


# items Method 

for i in data.items():
     print(i)

Output :-

# keys
Name                                                                                                                          
Age     
     
# values                                                                                                                 
Danish Ali                                                                                                                    
17 

#items method

('Name', 'Danish Ali')                                                                                                        
('Age', 17)

Aap Upper Diye gye output ko dekh kar program ko samanjh sakte hai.

python Dictionary in keyword

Agar Hame Ye Pata Karna Ho Ki Hamara Key & Value Dictionary Me hai Ya nhi to ye is example 6 me dekhte hai using in keyword |

Example 6:-

data  = {'Name' : 'Danish Ali' , 'Age' : 17 }

if 'Name' in data:            # find key 
      print('present')
else:
     print('Not Present')

if 'Danish Ali' in data.values():     # find value          
      print('present')
else:
     print('Not Present')

Aap Upper Diye Gye Program ko samanjh sakte hai.

Output :-

present
present

Update Method In Dictionary In Python

Update Method 2 Dictionary ko add karke ek bana deta hai , Aap Example 7 dekh sakte hai |

Example 7:-

data1  = {'Name' : 'Danish Ali' , 'Age' : 17 }
data2  = {'Fav Movie ' : '3 idiots ' , 'Sport ' : 'Crickt' }

# update mathod

data1.update(data2)

print(data1)

Aap Upper Diye Gye Program ko samanjh sakte hai.

Output :-

{'Name': 'Danish Ali', 'Fav Movie ': '3 idiots ', 'Age': 17, 'Sport ': 'Crickt'}

Aap upper Diye gye Updating elements ke output ko dekh kar program ko samanjh sakte hai |

python dictionary pop method

Python Pop Function :- Pop Function dictionary Me se ek Elements delete karta hai |

Example 9:-

data  = {'Name' : 'Danish Ali' , 'Age' : 17 }

remove_pop = data.pop('Age')

print('Remove Data Using Pop Function is {} '.format(remove_pop))

Aap Upper Diye Gye Program ko samanjh sakte hai.

Output :-

17

Aap upper Diye gye Pop Function ke output ko dekh kar program ko samanjh sakte hai |

Get () Method in Dictionary

get() method ham dictionary me isliye istemal karte hai kyoki agar hame koi dictionary me se value print karani ho bina error ke |

data  = {'Name' : 'Danish Ali' , 'Age' : 17 }

print(get_data)
